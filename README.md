### **# Example Body Array PHP**

**PHP Example Send Text :**
```php
$body = array(
    "api_key" => "xxxxxxxxxxxx",
    "receiver" => "6285xxx",
    "data" => array("message" => "Hello World")
);
```
**PHP Example Send Image :**
```php
$body = array(
    "api_key" => "b97b74f802c2802b94358267c160bbd282f5c384",
    "receiver" => "6285xxx",

        "url" => "https://i.ibb.co/QbmsBqs/code.png",
```
